# okeytour-next

Next.js (App Router) + TypeScript версия лендинга.

## Dev

```sh
cd okeytour-next
npm i
npm run dev
```

Откроется на `http://localhost:8080`.

## Что внутри

- Next.js App Router (`src/app`)
- Tailwind + shadcn/ui (`src/components/ui`)
- Providers: theme (next-themes), react-query, toaster/sonner (`src/app/providers.tsx`)
